i.What's the difference between of Local Storage and Sesion Storage?

Ans:- The difference between sessionStorage and localStorage is that localStorage data does not expire, whereas sessionStorage data is cleared when the page session ends.


ii.What's the difference between of Block Scope and Global Scope?

Ans:-Block Scope's data can access just inside the block scope area. On the Other hand Global Scope's data can access any where insider the Javascript file.
